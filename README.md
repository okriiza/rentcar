## How To Used

1. Clone this project, after clone run this command
    ```bash
    composer update
    ```
2. npm install
    ```bash
    npm install
    ```
3. key:generate
    ```bash
    php artisan key:generate
    ```
4. set your database
   
5. run migration
   ```bash
   php artisan migrate --seed
   ```
6. Run NPM run Dev
    ```bash
    npm run dev
    ```
7. run artisan server
    ```bash
    php artisan serve
    ```
8. login

   password: password
